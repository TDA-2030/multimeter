# WiFi Multimeter

A mini multimeter based on ESP32, that display the result on web page.

## Features

- Web page as monitor

- Portable, small size

- Voice announcements



## Functions

  - voltage: ±400V, ±60V, ±2V

  - current: ±10A, ±2A, ±10mA

  - resistance: 1KΩ, 200KΩ, 10MΩ

  - buzzer

  - power
  
  - diode drop voltage
